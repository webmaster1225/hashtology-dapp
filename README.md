# hashtology-dapp

This app is currently in development...

Main functionality of the app is to transfer Ethereum tokens from one address to another address

Techs used:

  > TypeScript

  > ReactJS | Vite 

  > TailwindCSS

  > Ethereum
 
  > Polygon | Mumbai 

  > Hardhat

  > EthersJs
  
  >  Alchamy 

  >  Metamask
